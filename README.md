#惜时 
1，时光一去不复返，珍惜时间，合理安排时间，就等于节约时间。 
2，直视时间分分秒秒流逝，说不出的心痛。 
3，一次只做一件事，记录耗时，可累计，珍惜时间，专注做好每一件事。

惜时和番茄工作法很类似，但宽松很多，允许自由设定时间及暂停，强调专注一件事，应该再加个滴答声，让人神经紧绷
